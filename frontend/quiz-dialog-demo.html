üß™ How to Test the Component

Method 1: Add to an Existing Page (Recommended)

Add the QuizDialog to one of your existing pages (e.g., Translate or Learn):

// In pages/Translate.tsx or pages/Learn.tsx
import { QuizDialog } from '@/components/QuizDialog'
import { useState } from 'react'

// Inside your component:
const [showQuiz, setShowQuiz] = useState(false)

const sampleQuiz = {
  quiz_attempt_id: 1,
  question: "What is the English translation of 'Katze'?",
  options: ["cat", "dog", "house", "tree"],
  question_type: "multiple_choice_target",
  phrase_id: 123,
}

// Add a test button somewhere in your JSX:
<Button onClick={() => setShowQuiz(true)}>Test Quiz</Button>

// Add the dialog at the end of your component:
<QuizDialog
  open={showQuiz}
  onOpenChange={setShowQuiz}
  quizData={sampleQuiz}
  onSubmit={(answer) => {
    console.log("Answer:", answer)
    setShowQuiz(false)
  }}
  onSkip={() => setShowQuiz(false)}
/>

Method 2: Test in Browser DevTools

You can directly test the responsive design without adding code:

  1. Once you add it to a page (Method 1), click the test button
  2. Press F12 to open DevTools
  3. Click the device icon or press Cmd+Shift+M (Mac) / Ctrl+Shift+M (Windows)
  4. Select devices: iPhone SE, iPad, or Responsive
  5. Verify layout adapts correctly
